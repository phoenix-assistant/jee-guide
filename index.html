JAC Delhi — DTU / NSUT / IIIT-Delhi / IGDTUW

GOVT Joint Admission Committee for Delhi govt-funded universities.

⚠️ Eligibility: 75% Class 12 OR top 20 percentile of board. With 72%, check your board's 20th percentile. CBSE's is ~72–75% — very tight.

⚠️ Delhi quota: 85% seats for Delhi domicile. OSD (Outside Delhi) gets 15% — cutoffs much higher.

📋 Admission Process

  1. Register on JAC Delhi portal (June–July typically)
  2. JEE Main rank used for merit — no separate exam
  3. Online choice filling → counselling rounds (4–5 rounds)
  4. Document verification + fee payment after allotment

Dependencies: Valid JEE Main rank + board % eligibility + domicile certificate (for Delhi quota)

IPU — GGSIPU (Guru Gobind Singh Indraprastha University)

STATE GOVT No 75% board requirement. 60% aggregate sufficient. Best govt option at this profile.

📋 Admission Process

  1. IPU CET — university's own entrance exam (May–June). Primary admission mode.
  2. Some colleges accept JEE Main score directly — check each college's policy
  3. Online counselling (July–August) — 3-4 rounds
  4. 85% Delhi domicile quota / 15% outside Delhi

Dependencies: IPU CET registration (separate from JEE). ₹1,200 application fee. JEE score is secondary.

⏰ IPU CET registration typically opens March–April. Check immediately.

JIIT Noida — Jaypee Institute of IT

PRIVATE DEEMED NAAC A++ · Est. 2001 · Noida Sector 62 + 128

🔥 Best private option in NCR at this rank range. Strong placements (Amazon, Samsung, Adobe, Paytm). Modern campus. 38K JEE rank qualifies for scholarship negotiation.

📋 Admission Process

  1. Apply on JIIT website (online application)
  2. Merit based on JEE Main rank/percentile — primary criteria
  3. No separate entrance exam
  4. Direct admission offers with scholarship tiers based on JEE rank
  5. Confirm via fee payment (partial refundable if you withdraw by deadline)

Board %: No hard cutoff. Must have passed 12th with Physics, Chemistry, Maths.

Fees: ₹3.5L–₹4L/year (before scholarship). With 38K JEE: expect 10–25% scholarship.

⏰ Applications typically open Feb–July. Multiple rounds.
